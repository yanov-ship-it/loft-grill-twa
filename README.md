# Loft Grill TWA

This project is a Telegram WebApp for Loft Grill, providing an interface for users to interact with the services offered.

## Features
- User-friendly interface
- Admin dashboard
- Node.js backend

## Getting Started

1. Clone the repository:
   ```bash
   git clone https://github.com/yanov-ship-it/loft-grill-twa.git
   ```
2. Change directory:
   ```bash
   cd loft-grill-twa
   ```
3. Install dependencies:
   ```bash
   npm install
   ```
